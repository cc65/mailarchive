Hi all!

Welcome to the list. I have never before installed a list manager, or managed
a mailing list, so some things may not work as expected. If this happens,
please bear with me and send me a private mail.

I have placed version 2.1.1 of the compiler on the ftp server a few minutes
ago. This is just a bugfix release. It fixes all bugs listed on the "known
bugs" web page for version 2.1.0:

  * Two small problems with code generation.

  * An assert that is sometimes triggered in the assembler on the use of a
    command line defined symbol.

  * The sample programs were missing in the binary packages for DOS, OS/2
    and Windows. This created by far the most trouble. Several people mailed
    me, since they had no idea how to start.
