Just for information:

The problems with VICE described in the file debugging.txt are all gone in
version 1.0. The "ll" command does now work (no need to use "pb") and VICE 1.0
will correctly use and display all ca65 labels including static and local ones
(there are still lots of warnings regarding the values of labels that have
changed, you may safely ignore these). Debugging assembler programs with VICE
is now real fun:-)

There is still no source level debugging (and I don't know if this will ever
come), so when debugging C programs you need a good knowledge about the
compiler generated code, but it's very helpful anyway.
