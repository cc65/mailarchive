I've added support for predefined macro packages to ca65 today. Macro packages
are activated with the 

	.macpack	package-name

directive. There is currently one macro package available that defines long
conditional jumps (this package is used by the compiler). Others packages will
follow.

The changes (including docs for the new feature) are in the current source
snapshot. Comments and other feedback is appreciated.
