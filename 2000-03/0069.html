PLEASE note that the size limit for messages to the list is 40000 bytes. This
should be enough for even large messages. However, it is not enough for
binaries or large diffs. I don't think binaries or large diffs should go to
the list, because there are a lot of people on the list that may not want
these mails.

It is ok to offer files, but don't post them onto the list. Put them onto an
FTP server instead, and post the URL. If you don't have access to an FTP
server, you may send me the files in private mail, I will place them onto the
musoftware FTP server.

This time, I was not able to extract the attachment from the original mail,
since the list server destroys the necessary headers when creating the bounce
mail. So if you want to have a look at the mentioned diffs, please contact
Chris directly.
