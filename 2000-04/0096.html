I would like to add a new section to the cc65 homepage linking to projects
using cc65 in any way or building on it in some form. This would include any
sort of hardware that is known to work with cc65, programs developed using
cc65, any ports of cc65 to systems not currently supported, or any other
interesting use of cc65.

If you have any information about such projects, I would be glad if you could
drop me a mail.	If you are using cc65 for such a project and don't have a web
page, I would also be interested to hear from you, maybe I can at least add a
few sentences describing your project, so people get an impression what could
be done with cc65.

BTW: Has anyone tried Maciej Witkowiaks fine GEOS port? I'm not using GEOS
myself, but I have integrated most of the code into the development tree, and
it would be very helpful to get some feedback. cl65 needs some adaption to
handle the special GEOS header correctly, and the docs are not part of the
development package (please download the GEOSLib package to get access to the
docs), but this are the only omissions I know of.

Thanks a lot for your help!
