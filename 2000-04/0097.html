Hi,
  I don't have a web page describing my current project.  So I better give 
you some notes on the project.
  Our project is a handheld PDA come with a 2 level 160x80 LCD display.  The 
chip we are using is a MCU from SunPlus Taiwan (www.sunplus.com.tw), with 
internal ROM 128K byte and internal RAM 2K byte in size.  A 128K byte RAM 
chip is connected in external to store user data and for program data stack. 
  Internal clock speed is 3Hz.
