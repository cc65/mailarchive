Hello group!

I was reading in the ca65 manual about the fact that it is a one-pass
assembler and therefore it is difficult to add a listing feature:

> Consequently, the assembler is not able to produce a listing. This
> could be added by reading the source file a second time if a listing
> was requested, but I didn't see an urgent need for a listing, so this
> was not a disadvantage for me.

This would be a valuable debugging feature for me at least... does anyone
else have any thoughts?  I would be willing to do a little work on this,
unfortunately I have so many other projects that it might take a while so
I was wondering if anyone else has the knowledge/time/interest to
contribute.
