In the meantime, Maciej sent me a first C version of the GEOS resource
compiler. I have integrated the code into the main tree, and added support for
.grc (GEOS resource compiler) files to the cl65 utility. I have created a new
archive named cc65-sources-beta-2.5-02.tar.gz containing these additions. If
you don't use GEOS, you don't need the new package since nothing has changed
apart from the grc support.
