Hi!

There was some talk lately about the usefulness of a utility to extract
information from object files generated by the assembler. I have written such
a utility and would be glad for some feedback. So if you're able to compile
the sources, grab the snapshot archive and give it a try. The utility is named
"od65", it is located in src/od65. Use the --help command line switch to get a
short help, since there are currently no docs.

As said above, any feedback is greatly appreciated!
