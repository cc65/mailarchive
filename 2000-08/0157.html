There is (again) an OS/2 binary package of cc65 available:

	ftp://ftp.musoftware.de/pub/uz/cc65/cc65-os2-2.5.0.zip

The package contains version 2.5.0 of the compiler as all other currently
available binary packages.

Thomas Bretz (tbretz_at_uni-sw.gwdg.de) has told me that he has done a port of
VICE to OS/2, so maybe there is now more interest in an OS/2 version of cc65
than before. Compiling cc65 for OS/2 is not a problem, but there were almost
no downloads, so I have dropped it to save me some time. Making os2 binaries
available again is a test, and the version may be dropped again.
