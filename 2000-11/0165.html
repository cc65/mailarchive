Good evening!

There hasn't been much traffic on the list lately, I hope this will change
soon;-)

We're currently preparing a new compiler release. My todo list is still quite
long, but a release date this year should be possible.

For this upcoming release, I would like to ask for your help:

  * If you have any open issues, bugs or suggestions not already addressed,
    please tell me about it now.

  * Some testing is needed, so if you're able to translate the compiler from
    the sources, please give the current snapshot a try. The development page
    (http://www.cc65.org/develop.html) lists most (but not all) things that
    are new in the development version. Having some feedback (even a short
    "works for me") would be great.

  * One thing not mentioned on the development page is a module written by
    Christian Groessler to allow low level disk i/o in a fairly portable
    manner. The current implementation is for the Atari only. Any comments or
    test are very welcome.
	   
Thanks in advance for your help!

Regards


	Uz
