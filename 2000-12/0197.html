I'm mostly done with conversion/completion of the docs for the upcoming
compiler version, so I've placed the new files on the web for public review:

	http://www.cc65.org/doc-2.6/

The new documentation format is SGML (using the LinuxDoc DTD), so the link
above does point to easily browseable HTML files. This may be quite useful
even for people using version 2.5 of the compiler package.

If you find any problems with the docs (typos, omissions, ...), please drop me
a note. Thanks in advance!
