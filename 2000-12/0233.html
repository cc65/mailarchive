Today I downloaded CC65 for OS/2... but the binary archive is not complete...
Miss some headers (?) (es c64.h) and the libraries...

not the same!

I tried to modify gcc.mak files to compile all CC65 V 2.6 under OS/2
with EMX/GCC 0.9d for OS/2! I created emx.mak and now (with some Warnings)
I can use emx instead watcom...

Last step.... I successfull compiled the libraries under OS/2 without using linux and
without modify library makefiles... I used an sh.exe (a port of bash, but I don't
know the version) and the port of GNU Make (for OS/2)...
