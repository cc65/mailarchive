On Tue, Jan 02, 2001 at 06:42:02AM -0500, Mark Keates wrote:
> Out of the C64 emu's (Frodo,Vice,CCS64)
> which monitors do you prefer?

I prefer VICE for several reasons:

  * It emulates most of the Commodore machines, so I don't have to switch
    emulators for multi-platform code.

  * It has a very capable monitor, the only drawback being the arkward user
    interface.

  * It is available for and works very well under Linux.

  * The source is available, so I'm able to check the code if I have problems.

  * The developers are very responsive.
