Tiny update to my earlier message... I have written and used two
small programs using the zero page area 163-171 as storage for
a small array that gets accessed over and over. So far it is working
well. But if you do any file I/O, the first 3 address's can get over
written. If you are just loading and going like me then it seems
safe to use so far. More testing to do on the other two "large"
zp blocks I mentioned.
