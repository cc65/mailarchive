>On Thu, Feb 08, 2001 at 02:42:47AM +0100, Christian Groessler wrote:
>> I've modified the kermit65 sources to be assemble-able by ca65.

>Do you think it would be possible to make these sources compile/assemble on
>more than one platform? Are the sources very Atari specific?

Since the Atari version is ported from the C64 version (which in turn
was ported from the Apple version) there should be some amount of
common code.

I haven't yet compared the C64 to the Atari version, but I assume
(and reading from the changelog of the Atari version), that there are
still quite some differences (screen/floppy/rs232).

The reason I did the ca65 port was that I wanted to try some changes
and wasn't able to find an assembler which compiled the original
version.

But I will remember the cross-platform idea, and check the differences
btw. the ports when I find some time.
