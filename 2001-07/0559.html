DIO  (low-level  disk  I/O) is now supported for Atari and GEOS (libraries
from CVS). If anyone would like to write C64/128 support please look  into
GEOS  dio  sources  because  half  of  work  is  already  done  -  I  mean
logical<->physical sector number conversion for  15[478]1  drives.  Things
that  are  to  be  done  include only disk I/O part - checking drive type,
sending fastload/save code there and reading/writting sectors.
