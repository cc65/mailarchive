On Sun, Aug 12, 2001 at 03:34:52PM +0200, udo klasmeier wrote:
> File-IO geht ja laut FAQ nicht für C64.
> Lassen sich denn die Kernal-Functions open, close, usw.
> benutzen, oder sind die ausgeblendet?
> Könnte man mit denen File-IO machen ?

Please try to write in english. There are many people on the list that do not
speak german. 

The memory layout for all CBM machines with the exception of the 6509 family
(CBM 600/700) has the kernal ROM banked in. So you may use these functions
from your C code. Marc Rintsch has started to write C wrappers for the
Commodore file functions, but I don't exactly know if they are included in the
current stable version, and how far he got.
