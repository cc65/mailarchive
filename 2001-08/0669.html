On Wed, Aug 15, 2001 at 07:08:58PM -0400, Raj Wurttemberg wrote:
> I'm new to cc65 programming and not very familiar to some of the specific
> Commodore API's and commands that are used. Is there a web site or ftp site
> with programs written for the cc65 compiler.

I've once tried to collect a list of projects done with cc65/ca65, but there
not that many replies, and most of them had no web page or similar
representation on the net. I know that many people are playing with cc65, but
it seems the number of finished projects is not that large.

One thing I've written that is not distributed with the compiler is a morse
trainer software for the C64. It uses all three voices of the SID to simulate
background stations and background noise, and it has a small interrupt handler
written in assembler, so you can see how easy it is to mix asm and C. The
sources are available from the musoftware FTP server. I'm currently not at
home, so it's difficult to look up the URL, as far as I remember, it was

    ftp://ftp.musoftware.de/pub/uz/misc/morse-1.0.zip
