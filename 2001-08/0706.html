> The Program runs with no complains, it generates the file
> "printer.dump" but the file is empty.
>
> Any idea what an be wrong.

AFAIK "True Drive Emulation" has to be turned off if you use the 
printer emulation. And there's no way for VICE to know when you are 
finished with printing because a close doesn't show up at the IEC bus. 
Therefore you have to tell VICE explicity (?) to flush the file. It's 
somewhere in the printer settings ("Flush printer device" or similar).

This is more a VICE specific question...
