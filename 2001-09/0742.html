Winsock is almost the same of BSD (at least, it's compatible). The only
things you have to change is:

- #include "winsock.h" or "winsock2.h"
- use WSAStartup() and WSACleanup() for Initialization of network stuff
- link with wsock32.lib

If you encounter problems, write to me directly, I've done this conversion
more than once.
