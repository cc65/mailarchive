On Wednesday 17 October 2001 16:05, you wrote:
> On Wed, 17 Oct 2001, Adam Dunkels wrote:
> > Does the 1541 ROM have the ability to boot a program from the disk?
>
> Yes, when you issue the drive a "&" or "B-E" command.  You could boot up a
> short loader that loads the rest of the software from the disk.
>
> Earlier versions of the firmware would execute the "&*" command at
> power-up if the CLK and DATA lines were held low.  This functionality was
> removed from the 901229-05 firmware version.  BTW, does anyone have
> 901229-04?  It's missing from FUNET.

Ok, so some kind of manual intervension would be needed for the newer models. 
That's a shame, the ideal would be that no C64 would have to be used - only 
the 1541 drive and a floppy containing the web site/ftp archive.
