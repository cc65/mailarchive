Hi all!

I just wanted to announce that uIP version 0.6 just has been released. For 
those of you who does not know it, uIP is a very small implementation of the 
TCP/IP stack suitable for 8- and 16-bit systems. uIP is released under a 
BSD-style license and is free to use for commercial and non-commercial 
purposes. uIP is easily compiled with cc65 and has been running on Uz' C64 
web server since october at http://c64.cc65.org.

The new features in version 0.6 are ARP support for running uIP over 
Ethernet, the ability to open TCP connections from within uIP and an improved 
application/stack interface. The documentation has been thuroughly reworked 
for the new release and examples on how to write application programs for uIP 
has been added.

uIP has been ported to a number of different platforms including the 
C64/Atari (through cc65 library support), Hitachi H8S/2148, the 8051 and the 
Atmel AVR.

For more information about uIP, visit the uIP web page at 
http://dunkels.com/adam/uip/.

Since there only has been one post to this list since new years eve, I'll 
also take this opportunity to wish you all a happy new year!
