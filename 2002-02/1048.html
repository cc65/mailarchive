On Fri, Feb 01, 2002 at 10:27:52AM -0500, Keates, Mark wrote:
> Has anyone built a config file that could be used
> to generate a NES rom image during the link phase?

The sources do contain a linker config for the NES. 

There was once a guy who wanted to start a NES port. He gave up very early,
because someone else told him that what he wanted to try was impossible. After
some time I found out that from the few existing NES files, some where copies
of code from other people (without any attribution), and others were plain
wrong - things were coded in a way that even I could see that they will never
work. So I removed NES support completely.

If I remember correctly, it was me who wrote the NES linker config using
information this guy gave me. So you may give it a try, but be careful, I'm
pretty sure the config has never been tested.
