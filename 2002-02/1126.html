Also, there's the "n queens" puzzle (create a chess board of n by n 
elements and place n queens on it in such a way that no two queens are 
on the same vertical, horizontal or diagonal line).  I think I saw it in 
the "examples" directory of bcc (Bruce Evans' C compiler), a C compiler 
targeted for the 8086, available from 
<URL:http://www.cix.co.uk/~mayday/>.   The problem should be famous 
enough so that you could find benchmark data for it.
