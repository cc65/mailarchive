I suspect that the hardware adoption needed for the Atari would be quite 
small. I haven't got a clue how the Ataris work, but I'm guessing based on

decoder chip.

easy - it should just be a matter of changing address offsets. The CS8900a 
code is contained in the cs8900a.c file in the C64 server source code.
