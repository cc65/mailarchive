On Mon, 18 Nov 2002, Ullrich von Bassewitz wrote:

> Does anybody know the maximum file name lengths for these platforms:
>
>         Atari

DOS standard: 8.3 (i.e. FILENAME.EXT).

>         <platform>-<width>-<height>-<colors>.tgi

Not all Atari DOSes allow '-' in filename.

Here's a Perl regex: :-)
[A-Z][A-Z0-9_]{0,7}(\.[A-Z0-9_]{0,3})?
