It seems that for the Commodore machines, the lowest common denominator would
be a 256 byte window supplied by the driver. The program may issue requests to
map part of the extended memory into this window. The GEO-RAM cartridge
supports this type of memory. I don't know about memory extensions on other
machines, however.
