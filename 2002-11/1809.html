>>>>> "UvB" == Ullrich von Bassewitz <uz_at_musoftware.de> writes:

UvB> [Scratching my head] Free RAM for what? The C128 has 41K memory
UvB> available when writing programs using the default cc65 setup.

Compared to the 50K in C64 mode, and even that's going to be a tight
fit for this project:

	$0000	system
	$0400	tables
	$0800	code
	$c000	screen ram
	$c400	sprite cache
	$d000	fixed sprites
	$e000	bitmap
	$ff40	mouse pointers

Then SuperRAM is used for storing sprites, tile data, and samples.
