BTW, here is a cc65 specific suggestion for your code: Pointers like "item"
are ideal candidates for the "register" keyword. It does no harm when used
with older compiler versions, so the code will still compile with 2.8.0.
However, using "register" in the code and -Or (or better) on the command line
will decrease code size and increase speed.
