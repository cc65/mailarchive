I don't know if this is of any help, but although the most recent versions
of GCC don't seem to support the Amiga any more, there exist patches that
allow the compilation of an m68k-amigaos targetted GCC 2.95.  See
<URL:http://www.lysator.liu.se/~lcs/files/gg-cross/> and possibly other
places.  Using that compiler, I've compiled AmigaOS programs on a
Pentium 3 system running GNU/Linux.
