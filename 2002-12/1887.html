You prefer a native compiler to compile a cross compiler?

Anyway, having done a bit of development with UAE (AGS on my arcade
home page), I can't say I can recommend it. The Picasso96 emulation
(gives the emulated amiga access to the PC's graphics card resolu-
tions) is particularly unstable and kept crashing the machine when I
flipped between screens (CED -> WB -> App). Flipping from UAE to the
Windows desktop would also crash the system half the time.

I'd follow Marko's advice and crosscompile with gcc and only use UAE
for testing.
