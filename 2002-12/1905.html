> mmmmh using VDC mem in c64 mode would be major cool.... that would finally be
> a reason to use that one and only acceptable c64 emulator :o))))))))

I think, Maciej is going to write a driver for the VDC memory. I will write a
driver for the additional memory of the CBM-II series. A memory extension for
the hidden memory of the C16 doesn't make sense: instead of using extended
memory, one could also use the plus4 target which makes the memory available
as linear memory. This means, that drivers for the most common memory
extensions for the Commodore machines are available or will be available soon.

If anyone would be willing to write drivers for the other machines, this would
be really great!
