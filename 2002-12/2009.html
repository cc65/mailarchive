> The left arrow symbol is the standard symbol for escape, so I would say this
> key *is* the escape key for the C64 and VIC20. 

Just checked this: Several of my PETs have the left arrow symbol on the key
but generate 27 as code for this key, so I would say the left arrow key (the
one in the upper left corner) is definitely the ESC key. I'm wondering
however, why this key generates code 0x03 for the C64 (and VIC20).
