Hi!

To test my packaging script[*] for the second tier operating systems, I've
created ZIP archives for Windows, DOS and OS/2 using the current sources
(which are roughly equal to the snapshot from last night).

If you are a user of one of these operating systems, please give the packages
a try and tell me about any inconsistencies as quickly as possible (missing
files, errors, other problems). Without further notice, the upcoming release
will use exactly the same packaging script[*], so any problems that are in
these packages will also be in the final 2.9.0 packages. And, by the way, this
is chance to get an update for a 2.8.0 installation:-)

The prerelease packages can be found in the snapshot directory of the FTP
server:

        ftp://ftp.musoftware.de/pub/uz/cc65/snapshot/

The usual disclaimer about the FTP server applies. The names used are somewhat
different from the names of the RPM packages in the same directory because of
limitations in the windows script.[*]

        cc65-apple2-snapshot-2.8.5.20021213.zip
        cc65-atari-snapshot-2.8.5.20021213.zip
        cc65-c128-snapshot-2.8.5.20021213.zip
        cc65-c16-snapshot-2.8.5.20021213.zip
        cc65-c64-snapshot-2.8.5.20021213.zip
        cc65-cbm510-snapshot-2.8.5.20021213.zip
        cc65-cbm610-snapshot-2.8.5.20021213.zip
        cc65-dos32-snapshot-2.8.5.20021213.zip
        cc65-geos-snapshot-2.8.5.20021213.zip
        cc65-os2-snapshot-2.8.5.20021213.zip
        cc65-pet-snapshot-2.8.5.20021213.zip
        cc65-plus4-snapshot-2.8.5.20021213.zip
        cc65-vic20-snapshot-2.8.5.20021213.zip
        cc65-win32-snapshot-2.8.5.20021213.zip

Please note that as with the standard release, you need an archive for your
host operating system and one or more archives for the target systems. All ZIP
files must be unpacked in the same directory.

Regards


        Uz


[*] Using the word "script" for something on a windows box is an exaggeration
that makes me smile each time I'm writing it down. In fact, it's a very
limited batch file with lots of ugly Windowisms.
