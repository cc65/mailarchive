It will include all stuff you got so far (grc as VLIR linker and ld65 config
generator as well as documentation updates and example for doing assembler
VLIR application) but except map_cc65 tool. That tool worked for me, but so
far I didn't have time to review your bug report. I don't even know if it
should be distributed as a part of cc65 package.

map_cc65 (name was chosen randomly) takes VICE label list and verbose
map (both from ld65) as input and uses them to create a list for including in
ca65 asm projects. It seems to have some problems on Win32.
