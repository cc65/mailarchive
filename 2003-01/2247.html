I use an editor written by the guy (Shawn Hargreaves) who developed the game programming library Allegro.  It is called FED (Folding Editor) and has all sorts of nifty functions that help in programming, not to mention the ability to "fold" whole functions or blocks to get them out of your way.  There are linux and windows/dos versions.  I use the dos version of course, so can't tell you how well the linux version works.

It isn't really a fully fledged IDE, but has all some very powerful features once you get used to it.

http://www.talula.demon.co.uk/fed/
