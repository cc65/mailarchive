Hi,

Three best editors for Windows I know are:
- Edit Plus (www.editplus.com)
- ConTEXT (www.fixedsys.com/context)
- Code-Genie (www.code-genie.com)
All can be configured for 6502 mnemonics highlighting and have no problems
with stripping file extensions.

There are syntax definitions for all these editors for xasm (my 6502
cross-assembler, http://xasm.atari.org) - it's easy to adapt them for
ca65.
