It's not the same, but it's effectively the same as doing flexible array

> Ok, this is something I have not thought of. In fact, I never even used it,

I'm still undecided about the initialization feature, but flexible array
members as defined by the C99 standard are now supported by the latest
snapshot. I've also made more than half of the changes needed to support
Microsoft style initialization, but when doing so found more inconsistencies,
so I'm not sure if this will get finished. If you have a use for the C99
feature, please grab the latest snapshot and give it a try. Feedback is
welcome as always.
