I do already have two of these books (first print from 1984, and third/last
print from 1986). The book is ok (if someone is interested), but nothing to
get mad about. The disassembly is generated with an automated disassembler,
which adds a short comment for each subroutine function called and for each
zeropage cell used, so it's easy to see how the code works. However, there are
no manual comments, discussing the code in more detail (like in the "C64
intern" books for example).

CBM610 support for cc65 was done using my own disassembly which is available
at

	ftp://ftp.musoftware.de/pub/uz/cbm610/kernal610-orig.zip

(beware: comments are in german).
