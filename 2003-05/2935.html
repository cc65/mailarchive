Hi!

As an update for the Apple ][ discussion: Stefan Haubenthal has contributed
joystick code and a new character type classification table for the Apple. I
have converted the joystick code into a loadable driver and may have
introduced some errors, so I would be glad if anyone could test it.

Note: Since the apple has no file i/o, you will have to convert the driver
using the object conversion utility and link it statically to your
application.

This extension brings up a question: What type of file names does the Apple ][
support? I will have to invent a name for the driver that is actually usable
once file support for the Apple is there.

The new code is in the snapshot that will be available in a few minutes.
