>>>>> "UvB" == Ullrich von Bassewitz <uz_at_musoftware.de> writes:

UvB> The debug info file is already available in 2.9.x using the
UvB> --dbgfile option to the linker. The format is currently
UvB> undocumented, but will become "official" for the next version.
UvB> The VICE label file will be removed, instead more information
UvB> will be written to the debug info file. The goal is to have just
UvB> one file containing meta information about the input in an easily
UvB> parseable format. The debug info file will contain more
UvB> information, like information about segments, so a converter
UvB> (which is easy to write using some Perl magic) can not only
UvB> extract the labels, but also information about readonly segments,
UvB> and convert this into a VICE macro file. It is also possible to
UvB> add such a converter to the cc65 package.

While I use -Ln all the time, I see your point here. Will you include
a dbg2vice converter or do you want help with that?
