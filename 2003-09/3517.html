Hi!

Contiki version 1.1-rc0 is out! Contiki is a cross-platform, open
source, Internet-enabled operating system and desktop environment for
the unexpanded Commodore 64 and for a variety of other platforms.
Contiki includes a graphical windowing system, TCP/IP networking, a
multi-tasking kernel, a web server, a web browser, and a bunch of other
goodies:

		   http://dunkels.com/adam/contiki/

The C64 version of Contiki is intended for cc65, and compiles with
version 2.9.2 of cc65. This version is a "release candidate" of the
upcoming 1.1 release, which means that not all features of the 1.1
version is in there, but that most of the main functionality is in
place. 

The most notable changes from the previous Contiki version 1.0 are:

Base system:
 * Support for loadable programs
 * Dynamically loadable networking drivers
 * Dynamically loadable GUI themes
 * Configurable screen savers
 * Mouse pointer support (currently only with joystick driver)
 * Networking configuration is now saved on disk
 * 1541 disk speeder

Applications:
 * New and improved web server with ability to load individual disk
   files
 * New command shell application
 * "Directory" application which reads and displays all programs on a
   disk
 * "Web downloader" application that can download D64 images directly
   to a 1541 disk
 * The web browser can automatically start the web downloader
   application when a binary file is loaded
 * Several HTML and rendering bugs fixed in the web browser
 * The web browser now starts at the Contiki web portal,
   http://contiki.c64.org/
 * A new "plasma" screensaver added

Network device drivers:
 * Driver for the RR-Net Ethernet card
 * SLIP driver for the SilverSurfer card

... plus a bunch of bugfixes and some internal reorganization because
of the new features.

This is the same version that is shipped with the RetroReplay/RR-Net
networking bundle sold by Jens Schoenfeld, so those who have got that
already has version 1.1-rc0 on a 5.25" disk.

Many thanks to everyone who made this happen, especially to all the
friendly people on the contiki-developers mailing list!

Both a D64 binary version and the full source code can be downloaded
from the Contiki web page:

		   http://dunkels.com/adam/contiki/
