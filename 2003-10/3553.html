Some time ago I submitted this bug report:

#include <stdio.h>
void main() {
	int a = -0x5000;
	puts(a < 0x5000 ? "ok" : "error");
	getchar();
}

As I see, it still displays "error", even with optimizations disabled.
I think that at least the docs should explain that.
