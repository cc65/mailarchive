On Tue, Oct 28, 2003 at 10:55:19AM -0600, Geoffrey Wossum wrote:
> I'd like to use cc65 to develop code for a custom hardware target.  Is there
> any guide or HOWTO for porting cc65 to a new target?

Not really. However, reading the existing docs is a good starting point.

> The main task seems to be creating a new crt0 startup, and a memory map /
> linker configuration.  Is my best bet just to use an existing platform as a
> guide?

Yes. I would suggest using/modifying the linker config and startup code for
one of the simpler platforms, for example the C16. If your code is in the ROM,
you will also have to copy the initialized data from ROM to RAM. For most
setups, the copydata routine (in common/copydata.s) will do the job, but you
will have to call it from the startup code.
