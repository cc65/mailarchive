Hi!

After some test using wine I'm now able to crosscompile the DOS, OS/2 and
Windows versions under Linux. This means that I'm now able to produce nightly
builds of the development snapshots for these operating systems.

As a result, you will find lots of new ZIP files in the snapshot directory:

	ftp://ftp.musoftware.de/pub/uz/cc65/snapshot/

As with the RPMs, documentation files are now in a separate archive, so people
downloading DOS and Windows archives get the files only once.

If you are one of the few remaining people not using free operating systems,
and brave enough to give these snapshots a try, please let me hear if they
worked for you.

A few notes:

    * The ZIP files are for DOS/Windows/OS2 users, while the RPMs are for
      RedHat Linux (may work for other Linux distributions, too).

    * If you have more than one version installed, please be careful with
      the paths.

    * If you install one version over another, please remember to delete the
      old files first, otherwise weird things may happen.

    * When reporting bugs, it is even more important than before to tell me
      which version you're using.

Have fun!

Regards


	Uz
