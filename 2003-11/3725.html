On Mon, Nov 03, 2003 at 08:54:22PM +0100, Christian Groessler wrote:
> Cool. So you're running Watcom under Wine? 

Yes. There's a small problem with wine and XFree86 4.3: Wine complains not
being able to detect my keyboard, but since the compiler doesn't need any
keyboard input, there are just lots of messages to ignore. The error seems to
be known and will hopefully be fixed in the next wine release. BTW: I'm also
running the Texas Instruments C compiler for the TMS320 DSP under Linux (also
a set of Win32 applications). It is really nice to have one common development
environment that has the power of Unix under it's hood:-)

> May I ask which version of Watcom?

I'm using the last commercial version, 11.0b I think. I've been developing
software for DOS, OS/2 and Windows for a long time, so Watcom was the obvious
choice. OpenWatcom looks very similar, I assume that it shouldn't be difficult
to use it instead of version 11.0. Just have a look at the watcom.mak
makefiles, it is rather simple (the most difficult part was getting wine
running without problems).
