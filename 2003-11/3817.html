You asked this before, remember? scanf does not work with cc65 because the
existing file routines don't have ungetc(). So you will have to use sscanf
instead (note the different name). However, sscanf works with strings, so you
will have to read a line of input and then pass this line to sscanf. My last
mail had a small code snippet how to do that.
