Hello,

I'm proud to announce that whole cc65 package will be included in the
upcoming release of PLD-Linux 2.0 (Ac).

Packages for a number of architectures are already there:
ftp://ftp.pld-linux.org/dists/ac/ready/

and sooner or later will be there:
ftp://ftp.pld-linux.org/dists/ac/PLD/

PLD-Linux is a rpm-based distro that was derived from RH a long time ago.
Two things distinguish it from other such distributions:
- perfectly prepared and modular packages (a >1GB (uncompressed) live cd
  and a <60MB rescue cd were created from the very same package base)
- poldek - a tool easier to use and more powerful than apt-get for online
  (de)installation/upgrade of rpm packages (in fact, a statically linked
  poldek in a chroot environment is the only way to install pre-2.0 PLD)

The URL is http://www.pld-linux.org/ but guys are too busy with work on
packages to keep the page up to date and translate it :).
