On Sun, Dec 14, 2003 at 11:11:50PM +0100, Ullrich von Bassewitz wrote:
> Does anybody possess a Plus/4 with a RS232 adapter?

I have a plus/4 and a VIC-1011A (I think) RS-232 adapter, which fits in the
plus/4 user port if its case is removed.  The adapter was designed for the
VIC-20, and it also works in the C64/C128.

> Or, does anybody know how to build the necessary adapter?

Any adapter for the VIC-20 or C64 user port should work.  There are
instructions at http://www.funet.fi/pub/cbm/documents/projects/rs232/,
file rs232-userport.txt.
