(Only C files -- and assembly files that have character- and
string-literals -- need to be rebuilt for each cc65 target.  So, for
example, nothing in the "runtime" directory actually needs to be
re-assembled.)

You can remove "install"'s dependency on "all".  Change:

install:        all
        ...
        ...

to

install:        #all
        ...
        ...
