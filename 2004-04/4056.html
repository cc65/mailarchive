I use this *all* the time.  If it's holding a numeric value, I call it a
"local variable". :)  I also use it for state transition flags.  Note that
having your variable in zeropage takes 3 cycles for a read, and 3 cycles for
a write.  Doing the above uses 4 cycles for a write, and 2 for a read, which
is the same speed when constantly written & read, faster when there are more
reads than writes, and uses 1 less zp byte and 1 more code byte.

A bit off-topic, but that really isn't very unconventional code. :)
