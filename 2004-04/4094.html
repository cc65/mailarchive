Since it looked like there were no more votes, I've added my code to the
repository and did the necessary porting. The syntax for command line
arguments is now

        RUN:REM arg1 "arg2: foo, bar" "" arg4

That is, arguments are separated by spaces. Support for command line arguments
is available for the following platforms:

        C64, C128, Plus/4, C16, VIC20

As a nice side effect, the code is shorter than before, so programs using
command line arguments are somewhat smaller.

If anyone wants to give it a try: A new snapshot is currently building.
