> Can you use these from a C program?  I looked in the
> docs., and also tried it, to no avail.
>
> I want to printf("%d", __BSS_LOAD__) or similiar.

Yes.  Remove the first underscore, declare them as external names, and use
their addresses.

#include <stdio.h>

extern unsigned char _BSS_LOAD__[];
extern void const _BSS_SIZE__;
#define __BSS_SIZE__ ((unsigned int)&_BSS_SIZE__)

int main(void) {
 printf("%p\n%u\n", _BSS_LOAD__, __BSS_SIZE__);
 return 0;
 }
