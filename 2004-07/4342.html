On Thu, Jul 29, 2004 at 09:50:09AM -0700, Shawn Jefferson wrote:
> Aren't all local variables put on the stack?

No:

void foo (void)
{
    static char bar;
    ...
}

bar is a local variable but placed in static memory. The compiler will do a
similar thing if told so by using

        #pragma staticlocals (on)

The #pragma will make all local variables static
