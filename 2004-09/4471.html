Well, actually, the "official" way to do that particular task is to use
the VIC-II's structure declaration:

#include <c64.h>

VIC.addr &= ~(unsigned char)0x02;       /* upper/graphics char. display */
VIC.addr |=  (unsigned char)0x02;       /* lower/upper char. display */
