Hi,

I created a small etch-a-sketch style application using the stdjoy and
tgi drivers. All the targets that supported these drivers compiled and
produced some executables.

I am a bit curious to know if this application really runs on the
platforms. The current lynx snapshot has a few bugs that I fixed and
will send them to Ullrich today.

If any of you own some real hardware (apple2, apple2enh, c128-vdc,
c128-vdc2, c64 or geos) I would like to know if the application
works as a etch-a-sketch.

The application is supposed to draw a red frame with a grey drawing
area. A black pen should draw pixels when you move the joystick.
Pressing fire while moving the stick erases the drawn pixels.

The cursor pixel should be blinking.

In case this works I believe that it would be super-cool to arrange
a programming competition to produce the best game that has to
cross-compile to all platforms supporting tgi and stdjoy. (And some
stdsound? driver.)
