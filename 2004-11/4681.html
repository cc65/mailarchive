I'm seriously contemplating building custom NAnt tasks
for the various tools in CC65.  If anyone is
interested, please let me know.  I typically don't
code in the style used for submitting code to
nantcontrib, but will do so if people will actually
use the tasks.

For those who may not know, NAnt is a tool that allows
you to create XML documents to build and execute build
scripts.  It's similar to Make, but a whole lot easier
to maintain and a lot more cross-platform.  NAnt runs
on Mono and the MS.Net Framework (all vesions), so
practically all operating systems are covered.  

On a side note, does anyone here use an IDE for
writing C and/or ASM for CC65?  I've used VS.Net's C++
editor before which worked OK, but is a bit overkill. 
Has anyone used Eclipse for pure C development?  Does
anyone know of a good text editor or IDE for Assembler
that contains programmable syntax highlighting,
possibly even with 65xx profiles already installed?

Thanks a bunch!
