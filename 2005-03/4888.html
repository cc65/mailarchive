The easiest way to fix that is probably to manually remove

  HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\cc65

from the registry. After that, the install should work again. There may be a
few error messages, because the menu entries do already exist, but you can
ignore them.

I will check that at least the uninstall.vbs file is available for install to
prevent that trap.
