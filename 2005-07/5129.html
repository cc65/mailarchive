The name of this symbol has been changed without changing the references in
crt0.s. If you're able to translate crt0.s yourself (rename the output file to
vic20.o and replace the one that comes with the snapshot), just replace any
occurance of __IRQFUNC_COUNT__ by __INTERRUPTOR_COUNT__. Or download the new
snapshot in a hour or so.
