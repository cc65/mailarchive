to mind is these packages:

lynx@kosh:~$ dpkg -l | grep sgml
ii  debiandoc-sgml 1.1.87         DebianDoc SGML DTD and formatting tools

parser outp
ii  sgml-base      1.26           SGML infrastructure and SGML catalog 
file su
ii  sgml-data      2.0.3          common SGML and XML data
ii  sgmlspl        1.03ii-31      SGMLS-based example Perl script for

lynx@kosh:~$ dpkg -l | grep zlib

ii  zlib1g-dev     1.2.2-4        compression library - development

but by installing all these to a Debian system you should get the things 
to compile
