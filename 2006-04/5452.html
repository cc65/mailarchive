On Sun, Apr 02, 2006 at 04:08:35PM +0200, Markus Stehr wrote:
> How do i tell cc65 that he should skip $2000 till $3000 so that i can
> put a charset in there?
> Is there a meta command for that?

There is no easy way to do that. You can use a changed linker config, but
doing this you're responsible for code placement yourself. You may need to
juggle segments or even place part of the code into different segments.

Depending on your program, it may be the simplest solution to let the code
start at $3000, and use the data between $800 and $2000 for startup code and
the DATA and RODATA segments. Or, if that is not possible, add the range to
the heap. But all this depends on your program and needs a changed linker
config.
