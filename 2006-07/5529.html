Hello,

I just put some Debian packages for CC65 online on my homepage. If
anyone is using Debian, he might want to give them a try. Just add the
lines 

deb     http://www.trikaliotis.net/download/debian/cc65/ binary/
deb-src http://www.trikaliotis.net/download/debian/cc65/ source/

into your /etc/apt/sources.list, run "aptitude update" (or "apt-get
update"), and you can install these packages via apt-get or aptitude.

Currently, the packages are based upon cc65 snapshot from 2006-06-06
(the latest one Uz's web server!)

The debianization was done by Michael Klein (cf.
http://www.lb.shuttle.de/puffin/), I only adjusted it for a more recent
cc65 version and for the newer Debian version.

These packages should work on Debian/Stable ("Sarge"), Debian/Testing
("Etch"), and Debian/SID. They MIGHT work on Ubuntu, too. I would be
thankful if someone could tell me if they work there.

I will have a look at the traffic this generates; if it is not too much,
I am planning to build newer packages on a regular basis. In this case,
I might even sign the packages. ;)

Regards,
