Hi all,

i'm working on a little game with modified charset.
I noticed CC65 is translating uppercase letters to
inverted uppercase chars (0xC0+) and lowercase
letters to (uninverted) uppercase chars (0x40+).

Would it be possible to make CC65 translate string
constants so that lowercase chars actually become
lowercase chars (0x00+)?

I'd like

unsigned char * data="Test";

int main()
{
   int x,y;
   for (x=0;x<16;x++)
     for (y=0;y<16;y++)
       (*(unsigned char*)(0x0400+x+(y+1)*40))=x+y*16;	
   (*(unsigned char*)(0x0400))=data[0];
   (*(unsigned char*)(0x0401))=data[1];
   (*(unsigned char*)(0x0402))=data[2];
   (*(unsigned char*)(0x0403))=data[3];
   while(1);
}

to print "Test" instead of "[T]EST"

thanks,
   Andreas
