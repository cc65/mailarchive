Hi Rich,

>  Anybody on this list use CC65 for Windows PC?  IE target platform is PC?

The cc65 is a cross compiler for targeting 6502 machines. So the
target platform can't be a PC.

> I don't know enough about TCP/IP to know what kind of commands can be
> sent/received, IE how does serverPC tell the web browser when it is
> done with the conversion.

As pointed out on the usenet a proxy server is the way to go. Even the
question of deciding which grphic format the browser knows/wants is
really spcefied:

See the HTTP 1.1 spec (http://www.ietf.org/rfc/rfc2616.txt) in chapter
'14.1 Accept'

A modern browser specifies something like:

text/html, application/xml;q=0.9, application/xhtml+xml, image/png,
image/jpeg, image/gif, image/x-xbitmap, */*;q=0.1

This should give you an idea...
