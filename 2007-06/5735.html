No. The cygwin "packages" are simply .tar.gz files. As far as I can
tell, there is no meta-information available.

Uz, BTW: Building deb packages is easy with the files in the debian/
directory of your source tarball if the user moves the directory to the
root of the sources (the debian build system requires this). Just move
it there, (optionally: change debian/changelog so the packages gets a
decent version number,) and call dpkg-buildpackage to get everything
done automatically.
