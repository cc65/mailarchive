I just mailed Uz the new files for
libsrc/lynx/kbhit.s
libsrc/lynx/cgetc.s

The old versions caused a lot of complaints and I fixed the ones I have 
added here about a year ago. These files have the right touch and feel 
for the Lynx "flabode"-keys. And they are already in use in the MegaPak 
vol 1 release.

So please put them in the snapshot you intend to freeze for 2.12 if it 
is not too late already.

I also noticed that the Lynx stdjoy-driver does not need to do anything 
for flip-screen functionality. That code should be removed completely. 
The current driver works perfectly if you run the init code once at 
startup and DON'T re-initialize the driver when you flip the screen.
