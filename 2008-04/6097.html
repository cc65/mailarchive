Yes. I found out that the Lynx swaps the joypad internally in hardware 
when you tell it to flip the display.

The correct INSTALL routine would be:

INSTALL:
lda #<JOY_ERR_OK
ldx #>JOY_ERR_OK
rts

After this change the macros like JOY_RIGHT() work correctly in all 
cases and the programmer does not have to write any extra code for 
supporting flipped displays for left-handed people.
