The packages are now all signed. My GPG key has the key, 0x2AF47E44,
the fingerprint is: ECC5 D8FD 6374 CFBD 42A1  9917 28E1 D6A4 2AF4 7E44.
It can be obtained from the usual key servers. You can give this key the
trusted status for APT by issuing:

  $ gpg -a --export 2AF47E44 | apt-key add -

as root on Debian, or with

  $ gpg -a --export 2AF47E44 | sudo apt-key add -

as normal user on Ubuntu, or on Debian if you have set up sudo
accordingly.

If you want to get the sources and verify my key, too, you must add my
key to your local GnuPG keychain:

  $ gpg --recv-key 2AF47E44
