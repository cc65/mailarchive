On Sat, Aug 16, 2008 at 11:06:16AM +0200, Stefan wrote:
> Can i call a fastcall function through a function pointer?

This is not really a cc65 but a C question. Please have a look in a C book to
read about function pointers.

Here is a samples that compiles:

    static void fastcall speedcode (char fadepos)
    {
    }

    static void fastcall createfadecode (void fastcall (*f)(char))
    {
    }

    int main (void)
    {
        unsigned speedcodelen = 256;
        void fastcall (*fadecodeadr)(char) =
                (void fastcall(*)(char)) malloc (speedcodelen);
        createfadecode (fadecodeadr);
    }
