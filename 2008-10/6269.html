It's not the compiler but the header files, that cause the problem. Reviewing
the cc65 header file shows that it isn't perfect either. Is it possible that
you try the following patch and tell me if at least the error message "The
compiler is broken" goes away?

---------------------------------------------------------------------------
Index: va_copy.h
===================================================================
--- va_copy.h   (revision 3893)
+++ va_copy.h   (working copy)
@@ -39,7 +39,7 @@


 /* No action if we're using a C99 compiler */
-#if (__STDC_VERSION__ < 199901)
+#if defined(__STDC_VERSION__) && (__STDC_VERSION__ < 199901)
