The following program :

 

#include <conio.h>

 

#define STRIG0 0x0284

#define STRIG1 0x0285

static unsigned char *mouse_port;

unsigned char mval=0;

 

void main( void )

{

  cprintf("mouse_port=%i\r\n",*((unsigned char *)(0x0284)) );

  cprintf("mouse_port=%i\r\n",*mouse_port);

  mval=*mouse_port; 

  cprintf("mouse_port=%i\r\n",mval);

}

 

 

Produces the following output :

 

mouse_port=1

mouse_port=225

mouse_port=225

 

My question is, since these all (should) refer to the same memory
location ( 0x0284 ), they should produce the same output. The correct
output value is '1', not 225. Is there some issue with doing indirect
loads ( de-referencing an unsigned  char * )?
