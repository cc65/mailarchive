On Donnerstag 23 April 2009, Carlos wrote:
> And all stuff i have heard about saving zero page registers, ... before
> setting the handler from cc65 is not required?

ofcourse if the routine running in interrupt uses those zeropage adresses, 
then you need to save (and restore) them in the handler. other than that 
there arent such restrictions if the handler is written in asm. (ofcourse, 
make sure whatever you do in it, that it doesnt affect the runtime 
environment, like the software stack, the heap, etc)
