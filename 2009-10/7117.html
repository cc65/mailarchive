Number four can be found here (number three didn't make it to the server).

        ftp://ftp.musoftware.de/pub/uz/cc65/snapshot/

It fixes code generation problems intruduced with -rc2 (my fault - I've been
to hasty), adds error messages if any of the combined assignment ops (+=, -=,
...) is used with an invalid right operand, and has complete docs for all TGI
functions (contributed by Karri Kaksonen, thanks a lot!).

Please give it a try and report any problems. Provided that there are no
problems, this will become version 2.13.0 in a few days.
