Template Creator for DOS is a DOS program that can create and organize  
templates.  It works on Win32 up to WinVista.  You can find it at 
_www.simtel.net_ (http://www.simtel.net) .  Simply do a search for  "Template Creator for 
DOS," and it should be the first hit.  I am  advertising it here because I 
have a few templates that may interest  someone.  They're mostly <1k long 
but may give others ideas to create  more templates.  They are as follows:
 
*    _EMPTY.BAT:    an empty batch  file
*    _EMPTY.S:    an empty ca65  source
*    65_64.BAT:    batch file to  use cl65 to compile a c64 program then 
run c64s.  Needs to be  debugged.  For now, remove the line to execute c64s.  
User must  modify.  
*    65_A2.BAT:    compile an Apple  2 program and run ApplePC.  Needs to 
be modified.  You may want to  remove the emulator running line.
*    65_COMP.BAT:    empty batch  file to compile a cc65 program one file 
at a time.
*    65_FLDS.C:    An empty file  for a Fields dialog.  Created for the c64 
but easy to modify.
*    65_MAIN1.C:    Empty main file  for a cc65 program.
*    65OPEN.C:    a module to  handle opening and saving of data files for 
cc65 and the c64 target.
 
If you want one or more of these files, contact me.  If  you wish to email 
me directly, please start the subject header with "To  Joseph:" because I 
share an email address with somebody  else.
