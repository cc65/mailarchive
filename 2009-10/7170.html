Hi,

Just fyi... Today Adam Dunkels updated the Contiki nightly build
system to 2.13.0 - thanks :-)

Now the cc65-based Contiki targets (currently apple2enh, c64, c128)
make use of the following new features:

- single env var setup (CC65_HOME)
- default mouse driver name
- switch to 80 column video (only apple2enh, c128)
- reference (-u _main) on linker cmdline
- stacksize setting on linker cmdline
- secondary linker config (only apple2enh)

The latter two allowed together to remove all custom linker configs
used with 2.12.0 thus simplifying future cc65 upgrades.

Best, Oliver
