And now I have the package compiled at:

 http://daniel.serpell.googlepages.com/cc65_2.13.0svn4389-1_amd64.deb

Some problems:

* Build dependencies are not ok, you need "groff" (for .txt docs),
  "texlive-latex-recommended" (for .dvi docs) and "texinfo" (for
  .info docs).

* The platform packages don't include new mouse/joystick/serial
  drivers, also all the ".tgi" files are not included.

* There is no packages for some platforms (apple2enh, lynks, atmos,
  nes, supervision).

* The documentation should be separated in a different package
  from the main compiler.

I corrected those ans made a new package, the .diff.gz and .dsc are in:
 http://daniel.serpell.googlepages.com/cc65_2.13.0svn4389-2.diff.gz
 http://daniel.serpell.googlepages.com/cc65_2.13.0svn4389-2.dsc

The chages file and the amd64 binary (the rest of the packages are
arch-independent) are:
 http://daniel.serpell.googlepages.com/cc65_2.13.0svn4389-2_source.changes
 http://daniel.serpell.googlepages.com/cc65_2.13.0svn4389-2_amd64.deb

All the packages compiled on Debian-4.0 AMD64.

Hope it helps,
