Also the Atari Lynx has very special screen allocation. It actually 
overlaps with the boot rom and Mikey and Suzy register space. The CPU 
does not see all the screen space. The Suzy sprite engine and the LCD 
display work partly in separate memory from the CPU.

This also means that it is much better to be able to hand tune this 
allocation than trying to deal with it using a program.
