Good evening!

I've assembled a small preview of the new TGI functions. It is available here:

    ftp://ftp.musoftware.de/pub/uz/cc65/contrib/tgi-preview-0.0.1.txt
    ftp://ftp.musoftware.de/pub/uz/cc65/contrib/tgi-preview-0.0.1.zip

The archive contains the source (compilable with todays head branch and the
snapshot that is currently building), and all necessary files to run the
program on the C64. The font files in the package are converted borland vector
font packages.

It's called a "preview" because new features are there, but they may be buggy,
big and slow. Especially the line clipper seems to be broken, so beware when
compiling the program for other platforms. If your screen dimensions are
smaller than 320x200, and the program hangs, you will have to modify the
scaling factors for the fonts to make them fit on the screen.

Have fun!


        Uz
