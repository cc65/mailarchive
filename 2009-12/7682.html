Just in case you haven't already noticed...

http://www.vandenbrande.com/wp/2009/06/breadbox64-a-twitter-client-for-the-c64/

It's the first 3rd-party real cc65 Contiki application I'm aware of.
In this context "real" means not just using the uIP TCP/IP stack but
using...

- the Contiki build system
- the Contiki process, thread and event model
- the cc65-Contiki loadable Ethernet drivers
- the cc65-Contiki IP configuration website

This is the very type of standalone application I had in mind for
cc65-Contiki 2.x :-)
