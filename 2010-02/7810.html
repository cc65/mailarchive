Attached are 2 more new extended memory drivers.

1- +60k, memory expansion that can map $1000-$FFFF from 'external' ram by using $D100 as a control register.

2- +256k, memory expansion like the +60k, except that it can also map in $0000-$0FFF and has 256K of memory to chose from.
