Hiya,

The attached file contains the following joystick drivers:

- c128-humjoy.s, c64dtv hummer type userport joystick adapter for c128
- c128-oemjoy.s, vic20 type userport joystick adapter for c128
- c128-petjoy.s, pet type userport joystick adapter for c128
- vic20-humjoy.s, c64dtv hummer type userport joystick adapter for vic20
- vic20-oemjoy.s, vic20 type userport joystick adapter for vic20
- vic20-petjoy.s, pet type userport joystick adapter for vic20
- cbm610-humj.s, c64dtv hummer type userport joystick adapter for cbm610
- cbm610-oemj.s, vic20 type userport joystick adapter for cbm610
- cbm610-petj.s, pet type userport joystick adapter for cbm610
- cbm610-ptvj.s, protovision type userport joystick adapter for cbm610

Signed, Marco van den Heuvel.
