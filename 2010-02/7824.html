Only joysticks 3+4 have been changed by the patch. One change fixes definitely
an error in the driver, the other two invert the output bits for ports 3+4. I
cannot test the changes, but assume they're correct.

All patches from Marco have now been applied, partially with some changes by
me. The drivers still need rework before adding them. If anybody is able and
willing to do a few tests, I would be glad.
  
C128 machines with RAM in banks 2+3 are very rare, so I will probably move the
current c128-ram driver with support for these banks into a separate driver
(c128-ram2) and restore the old bevahiour for c128-ram. This will make the
driver for the most common case smaller.
