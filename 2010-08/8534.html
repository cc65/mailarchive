On Mon, Aug 02, 2010 at 02:54:00AM -0500, Payton Byrd wrote:
> Instead of dynamically allocating arrays for the selected entries on the
> heap, I set the arrays to memory locations at $A100 and $A500 and I simply
> fill these with '\0' to initialize them instead of using free() and
> calloc().

You can also add arbitrary blocks of free memory to the heap using the
_heapadd function. Once done, these blocks become part of the heap, and malloc
will use them to allocate memory on users request.
