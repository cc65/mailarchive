>
> And what about importing files _into_ an .ATR file ?

unix2atr. It puts all files from a directory to the ATR file. I don't know right
now whether existing files in the ATR are preserved, but I don't think so.
To add a single file to an existing ATR you could write a wrapper script (extract
existing files to a temp dir, add new file to temp dir, create ATR from temp
dir).
