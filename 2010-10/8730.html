I think that this is what you are looking for: 
http://www.cs.tut.fi/~albert/Dev/puzip/ It has public source code, don't 
know if it is open source.

Also there is this: ftp://ftp.info-zip.org/pub/infozip/doc/readme the 
appnotes for infozip and 
http://www.pkware.com/documents/casestudies/APPNOTE.TXT This is the 
documentation for zip file format. DEFLATE (the most populat zip 
compressor) uses a a 32Kbyte dictionary witch means that 32Kbyte needs 
to be free in ram in case it is used.
