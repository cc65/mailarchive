When installing cc65 with the info pages the resulting make.info will conflict 
with the info page from the gnu make package

Gentoo ebuild is doing this now:
# solve conflict by renameing make.info to cc65make.info
sed -e "s:make.html:cc65make.html:g" -i doc/index.sgml
sed -e "s:make.sgml:cc65make.sgml:g" -i doc/Makefile
mv doc/make.sgml doc/cc65make.sgml

thx
Mario
