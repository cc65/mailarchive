I found the problem and the solution.  I had changed a local variable from a
pointer to a struct to an instance of the struct.  After reading some old
posts in this group I discovered that the struct was too big for the 256
byte local variable size limit.  I was able to make the variables static
since it's not a recursive function and that made the error go away.
