I wouldn't recommend using batch files for anyone using cc65.  If you just
want quick compiles, just use cl65 on the command line.  If you need complex
compiles you need to learn MAKE.  If you don't understand what MAKE does,
then you should do some research.  Learning MAKE was one of the most
important steps in me getting to where I could manage CBM-Command as a
project.
