Ullrich added a feature that will reduce the amount of space taken by string
literals in the rodata segment by using pointers to duplicate strings.  I
believe that this works per object file though, so that is something to keep
in mind when trying to find optimizations for space.  This is something that
could be (or maybe is?) documented better.
