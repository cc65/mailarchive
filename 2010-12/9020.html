If you do ever need to avoid overwriting, then you can "cheat":

% cd /ram
% cat do-nothing.c
int main(int argc, char **argv) {
    return 0;
    }
% cc -o do-nothing do-nothing.c
% ln do-nothing cc65
% ln do-nothing ca65
% ln do-nothing ld65
% PATH=/ram:$PATH cl65 -d do-nothing.c
cl65: Cannot remove temporary file `do-nothing.s': No such file or
directory
Executing: cc65 -d -t c64 do-nothing.c
Executing: ca65 -t c64 do-nothing.s
Executing: ld65 -t c64 -o do-nothing do-nothing.o c64.lib
