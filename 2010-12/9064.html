Another workaround:

readreg.s:

_readreg:
    lda the_register
    rts

foo.c:

uint8_t readreg(void);
/* or: void readreg(void); */

void function(void)
{
    readreg();
    /* do the rest */
}

should result in:

    jsr readreg

Yes, still overhead of JSR/RTS.
