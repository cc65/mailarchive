Has anybody written a tool that would automatically collect makefile
dependencies (.include/.incbin) from ca65 source files? I was thinking
about doing it but don't feel like reinventing the wheel if not
necessary. :)

For example if foo.s includes foo.h and incbins bar.bin, it would generate this:

foo.o: foo.s foo.h bar.bin

Eventually it would be nice to see this supported in ca65 as well,
similar to "gcc -MM", to handle those rare cases where people do stuff
like .include .sprintf( ... ).
