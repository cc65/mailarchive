Hi Uz,

> One solution is to introduce an option flag
> similar to that in the mouse drivers. One bit could notify the TGI kernel
> about the driver using fixed point scaling for the bitmap font.

If we go into the direction of driver metadata here then we might
think too about a minimal and maximal scaling factor supported by the
driver. My "oversimplified" Apple II driver could then set both to 1.0
to indicate that it can't scale at all. Other drivers might only be
able to scale up (> 1.0), but not scale down (< 1.0).
