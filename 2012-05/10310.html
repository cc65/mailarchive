Greg,

You never cease to amaze me with your expertise of the platform.  I am very
happy to say that I got the cart working with the guidance provided in your
email.  The only thing I had to change was the config line for the DATA
segment.  When you use run = RAM, it tries to run the code from the bottom
of RAM, not from the memory under the ROM (I verified this in the VICE
monitor).  By removing that flag everything is copasetic.
