On Fri, May 25, 2012 at 04:50:34PM -0500, Payton Byrd wrote:
>    How do I get cc65 to use a custom crt0.s file?  Do I have to recompile the
>    libraries? 

No. Just create your custom crt0.s file and link it to your application by
naming it on the command line. If there are libraries on the command line, the
replacement crt0.o should be listed before them. And make sure, the file
exports the __STARTUP__ symbol as the standard crt0 files do:

        .export         __STARTUP__ : absolute = 1      ; Mark as startup
