Hi,

I wanted to point out that the new sample program 'multidemo'
basically depends on three things:

1. disk directory i/o
2. overlay linker config
3. at least one extended memory driver

The Atari target offers the first and the second but lacks the third :-(

Looking around a bit it seems that there was/is a quite popular RAM
extension "standard" called 'XRAM' which seems to be what is supported
by the Atari800Win PLus emulator. Maybe an Atari guru feels motivated
by 'multidemo' to implement an Atari extended memory driver...

Regards,
