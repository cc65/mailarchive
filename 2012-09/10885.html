I tested your program in various DOS versions:

- DOS 2.0 : only two first entries shown
- DOS 2.5 : only two first entries shown
- DOSXL 2.3 : only two first entries shown
- MyDOS 4.5: full list
- BwDOS 1.3: full list
- SpartaDOS X 4.21: full list

So, newer ones work ok, old ones have a problem opening files when
reading the directory. I think that MyDOS and DOS 2.5 were the most
used ones, but DOS 2.5 is incompatible with high capacity drives. I
personally use BwDOS.
