Hi *,

Any thoughts about providing `slow()` and `fast()` functions for the C64 
target that switch the screen on/off and would set a C128 into 1 Mhz/2 
Mhz mode?

It is common to do this in computing intensive programs like compression 
software.  C128 users get some extra speed out of the C64 application 
and on the C64 it doesn't do any harm AFAIK.  Even there you get a small 
speed gain because the VIC doesn't stall the CPU any more for memory 
access.
